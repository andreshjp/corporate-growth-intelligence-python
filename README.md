# Corporate Growth & Market Efficiency Analysis 📊

### Project Overview
This project focuses on analyzing a dataset of over 8,000 global companies to identify market leaders, assess operational efficiency through revenue-per-employee metrics, and segment performance by industry and location (specifically focusing on major hubs like New York). 

As a Business Economist, I approached this data to extract actionable insights regarding corporate scale and profitability patterns.

### Key Objectives
* **Data Cleaning & Standardization:** Handled missing values, standardized column naming conventions, and managed multi-currency revenue formats.
* **Market Segmentation:** Isolated high-performing sectors and localized analysis for strategic business hubs.
* **Efficiency Benchmarking:** Created new performance indicators, such as "Revenue per Employee," to evaluate operational health beyond gross income.

### Tech Stack
* **Language:** Python
* **Libraries:** Pandas (Data manipulation), OS (File management).

### Analytical Highlights (Insights)
* **Localized Leadership:** Identified the top-tier companies in New York, providing a clear map of market dominance in the region.
* **Operational Efficiency:** Through the calculation of Revenue per Employee, the analysis revealed which industries utilize their human capital most effectively to generate value.
* **Data Integrity:** Implemented a robust cleaning pipeline to ensure that financial figures were comparable across different reporting standards.

### How to Run
1. Clone the repository.
2. Ensure you have `pandas` installed.
3. Run `scripts/main_analysis.py` to generate the processed datasets in the output folder.
